# post--misread-tsne
This project generates an interactive article about misreading t-SNE.

## Local Setup & Preview

### 1. Install Dependencies
```bash
npm install
```

### 2. Build the Project
Generating `docs/index.html` and copying assets:
```bash
npm run build
```

### 3. Local Preview
Start a local server to view the result:
```bash
npm run serve
```
Then open the provided URL (usually `http://localhost:3000`) in your browser.
